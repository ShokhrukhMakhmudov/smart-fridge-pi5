## 3. Параллельные потоки

```
┌─────────────────┐     ┌──────────────────┐     ┌──────────────────┐
│  MAIN THREAD    │     │ SERVER THREAD    │     │  TIMER THREAD    │
│  (CV-цикл)      │     │ (uvicorn loop)   │     │  (auto-close)    │
├─────────────────┤     ├──────────────────┤     ├──────────────────┤
│                 │     │                  │     │                  │
│ Camera.read()   │     │ FastAPI routes   │     │ threading.Timer  │
│      ↓          │     │      ↓           │     │      ↓           │
│ detector.detect │     │ /lock/open       │     │ Lock._auto_close │
│      ↓          │     │  └─▶ Lock.open() │     │   └─▶ GPIO HIGH  │
│ tracker.update  │     │      └─▶ Timer───┼────▶│                  │
│      ↓          │     │                  │     └──────────────────┘
│ crossing.proc   │     │ WS /ws           │
│      ↓          │     │  ↑ broadcast()   │
│ on_event ───────┼────▶│                  │
│                 │     │                  │
└─────────────────┘     └──────────────────┘
```

## 5. Жизненный цикл события "взяли продукт"

```
Время  │ Кадр │ Где трек         │ Что происходит
───────┼──────┼──────────────────┼────────────────────────────────────────
 t0    │  1   │ y = 280 (ниже)   │ Trek #7 создан, prev_cy = 280
 t1    │  2   │ y = 270          │ prev=280, cy=270, нет пересечения
 t2    │  3   │ y = 245 (=line)  │ ⚡ crossed_up: pending_up = 1
 t3    │  4   │ y = 230 (выше)   │     pending_up = 2
 t4    │  5   │ y = 215          │     pending_up = 3
 t5    │  6   │ y = 200          │     pending_up = 4
 t6    │  7   │ y = 185          │     pending_up = 5  → ПОДТВЕРЖДЕНО
       │      │                  │     ▶ event "taken"
       │      │                  │     ▶ in_cooldown = True
       │      │                  │
       │      │                  │  ┌────────────────────────┐
       │      │                  │  │ on_event callback      │
       │      │                  │  │   ↓                    │
       │      │                  │  │ WS broadcast:          │
       │      │                  │  │ {                      │
       │      │                  │  │   "type":"crossing",   │
       │      │                  │  │   "event":"taken",     │
       │      │                  │  │   "track_id":7,        │
       │      │                  │  │   "product":"cola",    │
       │      │                  │  │   "timestamp":"..."    │
       │      │                  │  │ }                      │
       │      │                  │  └────────────────────────┘
 t7+   │ ...  │ y < line - 30    │ Кулдаун снят, можно ловить
       │      │                  │ следующее пересечение
```

## 6. Подавление по руке

```
БЕЗ HandSuppressor:                  С HandSuppressor:
                                     
Кадр 1: продукт y=280                Кадр 1: продукт y=280, рука НЕТ
Кадр 2: рука загораживает            Кадр 2: рука ЕСТЬ → пропуск
        бокс прыгает: y=200 (!)              prev_cy остаётся = 280
Кадр 3: y=180                        Кадр 3: рука ЕСТЬ → пропуск
        ⚠ ЛОЖНОЕ "taken"             Кадр 4: рука ЕСТЬ → пропуск
                                     Кадр 5: рука ушла, y=270
                                             prev_cy=280, cy=270 → норм
```
